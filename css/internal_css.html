Internal CSS

Internal CSS bir HTML sayfasında bulunan başlıkları etkiler.
head etiketi içerisinde title etiketinin altına konumlandırılır.

style etiketi arasına kodlar yazılır
p{
color: red
}

sadece ilgilisayfada ki kodları etkiler

yukarıda ki örnekte hem internal css hemde inline css e örnekler verdik
head etiketi içerisinde style etiketi açıp internal css tanıladık
header etiketi içerisinde de sadece o etiketi etkileyen inline etiket tanımladık
