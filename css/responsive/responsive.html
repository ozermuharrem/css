Responsive Tasarım

Masa üstü için yapılan web sayfalarının tablet veya telefondan açıldığında tasarımda değişiliklere sebep olmaktadır. Responsive tasarım farklı px deki tarayıcılardan açılan web sitemizin aynı görseli yakalaya bilmek için kullanılır

1366 px pc
700 px tablet
400 px phone

Responsive bir web sayfası hazırlaya bilmek için @madia screen etiketini kullanı yoruz. Screen pc-tablet-telefon gibi tarayıcılar için kullanılmaktadır

Yanda ki resimde kullanım şekli görünmektedir.
Burada en fazla 800px olan pc tablet veya telefonda tarayıcıları için h1 etiketininin font size ını 16px yap.

Yanda ki resimde kullanım şekli görünmektedir.
Burada en fazla 800px olan pc tablet veya telefonda tarayıcıları için h1 etiketininin font size ını 12px yap.

önemli not: Bu yazılan reponsive kodu çalıştıra bilmek için head etiketi içerisinde mutlaka "< meta name="viewport" content="width-device-width" >" kodu yazılmalıdır.
